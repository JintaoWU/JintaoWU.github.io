1003-Contest

题意：n个学生，m个问题，所有人回答问题的次数差不能大于1，求最大答对期望。

分析：只有 10 个学生，状态压缩学生，dp[i][j] 表示第 i 个问题，状态为 j 的最大期望值。
dp[i][now|next]=max(dp[i][now|next],dp[i-1][s]+p[k+1][i]) 当now等于 (1<<n)-1 的时候now要变为0,意味着回答这个问题的人可以是任何人。

弦图判定

弦：连接环中不相邻的两个点的边。

弦图：一个无向图称为弦图当图中任意长度大于3的环都至少有一个弦。

团：满足G为关于V的完全图的图。

单纯点：设N(v)表示与点v相邻的点集。一个点称为单纯点当{v} + N(v)的诱导子图为一个团。

ZOJ 3812 We Need Medicine

题意：给 n 个物品，物品 i 有两个属性，wi和ti。有 m 个查询，查询包含两属性 mi 和 si，表示能否在
n 个物品种选取某几个使得 sum(wi)==mi && sum(ti)==si,可以的话输出方案，不能输出“No solution!”。
数据范围：{ (1 <= n <= 400),(1 <= m <= 400),(1 <= wi <= 50),(1 <= ti <= 200000),(1 <= mi <= 50),(1 <= si <= 200000)}

分析：动态规划，普通方法超时超内存，所以要优化。
用 dp[x] 表示 mi==x 时 si 的状态，用bitset优化，记录路径时也要优化，每10个物品分一组记录，输出路径用dfs

Codeforces-D. Beautiful numbers

题意：给定一个区间，求区间内满足条件的数字的个数。条件：数字 x 的每一位能被 x 整除。

分析：数位dp，由题意可知，只要该数字能整除 1 到 9的最小公倍数（2520），那么肯定可以整除 1 到 9所有的数字。dp[i][st][j] 表示前 i 位，每位含有数字的状态位 st ，对 2520取余的余数。

Ceforces D. Appleman and Tree

题意：给定一个棵树，树上每个节点不是黑色就是白色。删除 k 条边，使得构成的 k+1 个联通块，没个联通块只有一个黑色的点，求 k 条边的集合有多少种情况。

分析：树形dp，dp[u][0]表示以 u 为跟的不包含黑色点的方案数，dp[u][1]表示以 u 为跟，包含一个黑色点的方案数。
