SPOJ-375. Query on a tree

题意:

给定一棵含有n个节点的树，有两种操作。

  • CHANGE i ti (将第i条边的值变为ti)。
  • QUERY a b (求a到b路径上权值最大的边)。

分析：

树链剖分

  • siz[u]表示以u为根的子树的节点的个数。
  • top[u]表示所在链的端点。
  • son[u]表示u的子节点种siz值最大的点。
  • dep[u]表示u的深度。
  • tid[u]表示树变成链之后u的编号。
  • fa[u]表示u的父节点
  • u到v有一条边，我用dep值小的那个点的编号作为这条边的编号。
  • 当u和v在同一条链上是(top[u]==top[v])直接用线段树做即可，如果不在同一条链上，假设fu=fa[u],fv=fa[v],depu=dep[u],depv=dep[v],depu>depv先对u ---> fu这条链进行操作，直到u和v在同一条链上。
